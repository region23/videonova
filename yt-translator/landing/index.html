Возможности приложения

Videonova использует передовые технологии ИИ для перевода видео с одного языка на другой.

Скачивание видео

Приложение автоматически скачивает видео и аудио с YouTube используя yt-dlp.

Распознавание речи

Встроенная модель Whisper с высокой точностью расшифровывает аудио и создает субтитры.

Перевод текста

Полученные субтитры переводятся на выбранный язык с помощью OpenAI API, сохраняя временные метки.

Генерация озвучки

Переведенный текст озвучивается с помощью передовых технологий text-to-speech.

Интеграция в видео

Приложение объединяет переведенную аудиодорожку и субтитры с исходным видео.

Кроссплатформенность

Приложение работает на macOS и Windows благодаря Tauri (Rust + веб-технологии).

Как это работает

Процесс перевода видео происходит в несколько этапов:

1

Вставьте ссылку на YouTube

Просто скопируйте и вставьте URL видео с YouTube, которое вы хотите перевести.

2

Выберите языки

Укажите исходный язык видео и язык, на который вы хотите его перевести.

3

Начните процесс

Нажмите кнопку "Перевести" и приложение автоматически выполнит все необходимые этапы.

4

Получите результат

Готовое видео с переведенной аудиодорожкой и субтитрами будет сохранено в указанной папке.
